Для сборки и запуска программы у Вас должен быть установлен maven  а также java версии 1.8.
После клонирования с github открыть терминал в корневом каталоге проекта Example,
далее набрать команду mvn clean package -DskipTests.
После данной команды будут доступны запуск и тестирование программы.
Для запуска набрать команду также в корневом каталоге java -jar ./target/example-0.0.1-SNAPSHOT.jar
Для запуска теста набрать mvn test.

В данной программе использована база данных H2, так что никаких дополнительных настроек для БД не нужно.
